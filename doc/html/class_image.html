Description détaillée

Classe représentant une image.

La classe représente une image de dimension dimx et dimy. La couleur de chaque pixel est stocké dans un tableau de type Pixel.

Constructeur avec dimensions de l'image en paramètre.

Paramètres
[in]dimensionX: entier positif, taille de l'image sur l'axe X (horizontal).

Créer une Image de dimensions (dimensionX , dimensionY) entièrement noir car appel du constructeur par défaut de la classe Pixel qui créer un Pixel noir.

Dessine un rectangle plein de la couleur dans l'image.

Tous les pixels se trouvant entre les coordonnées données sont remplis de la couleur du Pixel passé en paramètre.

[in]Xmin: entier positif, coordonnée minimum du rectangle sur l'axe X (horizontal).

[in]Xmax: entier positif, coordonnée maximum du rectangle sur l'axe X (horizontal).
[in]Ymax: entier positif, coordonnée maximum du rectangle sur l'axe Y (vertical).
[in]couleur: Pixel, désigne le couleur du rectangle dessiné.

◆ sauver()

void Image::sauver ( const string &  filename) const

Sauvegarde l'image dans un fichier.

Possibilité de mettre un nom de fichier pour s'enregister dans le répertoire courant ou de mettre un chemin, relatif ou absolu. L'image est enregisté sous un format .pmm, pouvant s'ouvrir avec la fonction ouvrir().

Paramètres
[in]filename: chaine de charactères (string), nom et chemin du fichier dans lequel on veut sauvegarder l'image.
